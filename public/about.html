Hi, I’m Tim, a developer from Auckland, New Zealand, currently studying Computer Science at the University of Auckland.

I enjoy building real-time and multiplayer web applications, especially projects focused on collaboration, synchronization, and interactive systems. Most of my work revolves around JavaScript, Node.js, Socket.IO, and browser-based technologies.

I created co-code.net as an experimentive real-time collaborative programming platform where users can code together directly in the browser with live synchronization and browser-based Python execution powered by Pyodide.



A lot of the project focuses on:

  • real-time networking
  • low-latency synchronization
  • browser runtime systems
  • scalable multiplayer architecture
  • secure client/server communication

I’m especially interested in the technical side of multiplayer systems and how interactive experiences can be made fast, responsive, and scalable using web technologies.
